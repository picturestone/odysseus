Speichern

Speichere folgenden Code auf deinem PC ab. Beim Laden kann der Code wieder eingefügt werden und der aktuelle Stand wird wieder hergestellt.

Sobald in das Code-Feld geklickt wird kann mit strg+a alles markiert werden. Dann mit strg+c kopieren und irgendwo auf dem PC abspeichern (z.B. in einer Word-Datei).

Laden

Kopiere den gespeicherten Code in das nachfolgende Textfeld und klicke auf "Laden".

Achtung! Laden löscht den aktuellen Stand unwiderruflich!
